## メモ
- グリッドサイズに対してボールサイズが大きいほうが衝突処理がバグりにくい
- ボールの速さが小さいほうが衝突処理がバグりにくい

- 衝突判定が厳密でなくて、跳ね返らずに色を変えすぎることがあるが、それくらいのほうが見ていて楽しいのでよしとする
